Hashing¶

Hash function¶

$$\mathfrak{h}: X \to H$$
  • $\mathfrak{h}(x) = \mathfrak{h}(y)$ only if $x=y$.
  • given $h\in H$, it is not easy to find $x\in X$ such that $\mathfrak{h}(x) = h$.

Types¶

Key¶

$\mathcal{F}=\{ f_\theta \:\ \theta \in \Theta\}$

$\left(f_{\theta}\right) ^{-1} = f_{\theta^\prime}$ for some $\theta^\prime \in \Theta$

$\theta \in \Theta$ : key

Symmetric¶

Using $\theta\in \Theta$, one can construct $\theta^\prime \in \Theta$ such $\left(f_\theta\right)^{-1} = f_{\theta^\prime}$.
